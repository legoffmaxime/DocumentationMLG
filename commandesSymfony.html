Voici quelques commandes Symfony utiles (Featuring Gassim) :

  • composer create-project symfony/skeleton nom_du_projet Crée un nouveau projet Symfony à partir du squelette Symfony.

  • composer require symfony/apache-pack Ajoute le pack Apache pour configurer facilement votre application Symfony avec Apache.

  • composer require symfony/asset Ajoute le composant Asset pour gérer efficacement les ressources statiques comme les images et les fichiers CSS.

  • composer require --dev symfony/maker-bundle Ajoute le bundle Maker pour générer rapidement des fichiers tels que des contrôleurs, entités et formulaires.

  • composer require symfony/http-foundation Ajoute le composant HttpFoundation pour manipuler les objets de requête et de réponse HTTP.

  • composer require --dev symfony/profiler-pack Ajoute le pack Profiler pour profiler et déboguer votre application Symfony.

  • composer require debug Installe le composant Debug pour ajouter des fonctions de débogage.

  • composer require symfony/orm-pack Ajoute le pack ORM pour utiliser Doctrine ORM avec Symfony.

  • composer require fakerphp/faker Installe la bibliothèque Faker pour générer des données de test aléatoires.

  • composer require symfony/form Ajoute le composant Form pour créer et gérer facilement des formulaires.

  • php bin/console make:entity Génère une nouvelle entité Doctrine avec laquelle vous pouvez interagir dans votre application.

  • php bin/console make:controller Génère un nouveau contrôleur Symfony prêt à l'emploi.

  • php bin/console make:form Génère un formulaire Symfony en demandant quelques questions interactives.

  • php bin/console make:user Génère une nouvelle classe d'utilisateur avec des fonctionnalités de base.

  • php bin/console cache:clear Efface le cache de l'application Symfony, utile après des modifications majeures.
